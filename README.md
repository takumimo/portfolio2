# ウォーリー

## サイト概要
ユーザー同士が悩みを共有するサイト

### サイトテーマ
悩みをみんなで共有しよう

### テーマを選んだ理由
1人で悩みを抱えている人を助けたい。ひいては、自殺防止に貢献できるようなサイトにしたい。

### ターゲットユーザ
周りに悩みを話せない若者(10代、20代)

### 主な利用シーン
悩みを抱えているとき
